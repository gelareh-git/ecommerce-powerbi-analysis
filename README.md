# E-commerce Sales Analysis Dashboard 2025

## Project Overview
This project involves an end-to-end data analysis of sales records using Power BI. It transforms raw data into a professional, interactive dashboard to track business performance.

## Key Technical Achievements:
- Data Modeling: Implemented a Star Schema to establish relationships between sales, products, and customers.
- ETL Process: Performed data cleaning and transformation using Power Query.
- Interactive Visualizations: Created dynamic charts including Monthly Revenue, Sales by Category, and Daily Trends.
- DAX Calculations: Developed measures for total revenue and performance tracking.
